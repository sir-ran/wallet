# wallet

> 一个使用vue全家桶构件的web钱包记账应用

## 主要技术构成

> 主要技术栈为vue.js、vue-router、vuex、部分组件使用了vue第三方组件库Mint-ui，图表部分使用了echart.js

## 主要实现功能

> 添加资产、添加负债、按用途和来源记录每天收入支出、使用localStorage存储个人信息，图片显示个人资产各部分占比
